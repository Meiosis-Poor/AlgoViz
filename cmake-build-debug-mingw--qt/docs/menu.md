# 图算法模拟器整体介绍

图算法模拟器是一个集图编辑、算法演示和数据可视化于一体的交互式工具，旨在帮助用户理解图论算法及图结构操作。

---

## 1. 功能概览
1. **图编辑**
    - 支持插入、删除节点。
    - 支持添加、删除边，设置边权值。
    - 节点和边均可通过点击选中进行操作。
2. **图结构显示**
    - 提供邻接矩阵和邻接表两种显示模式，直观展示图数据。
    - 可在界面右侧查看图的完整结构。
3. **算法演示**
    - 支持 BFS、DFS、Prim、Kruskal、Dijkstra 等算法。
    - 动态显示算法执行过程，包括节点访问顺序、遍历边和路径。
    - 支持单步操作、回退上一步和自动播放。
4. **随机图生成**
    - 可生成指定节点数和边概率的随机图，方便测试和演示。
5. **文件操作**
    - 支持保存当前图到文件。
    - 支持加载图文件，自动识别邻接矩阵或邻接表。

---

## 2. 使用方法
1. **图操作**
    - 点击“Insert Node”增加节点。
    - 点击“Delete Node”删除指定节点。
    - 点击“Add Edge”或“Delete Edge”操作边，需先选中节点。
2. **切换显示模式**
    - 邻接矩阵模式显示二维矩阵。
    - 邻接表模式显示链表/数组形式。
3. **算法演示**
    - 选择算法后，点击“Next”执行下一步，点击“Last”回退上一步。
    - 点击“Auto”按钮可自动播放算法执行过程。
    - 调整“Speed”滑块控制播放速度。
    - 关键节点和边会用颜色高亮显示（红色当前访问、绿色已访问、黄色路径）。
4. **随机图生成**
    - 输入节点数量、边权值范围和边生成概率即可生成随机图。
5. **文件操作**
    - 点击“Save”保存图到文件。
    - 点击“Load”加载图文件，支持邻接矩阵或邻接表。

---

## 3. 可视化效果
- **节点**：红色表示当前访问节点，绿色表示已访问节点，黄色表示选中节点或路径节点。
- **边**：红色表示当前遍历边，绿色表示已访问边，黄色表示最短路径或选中路径。
- **算法过程**：通过动画显示节点访问顺序和边遍历，帮助理解算法逻辑。

---

## 4. 适用场景
- 学习与教学图论算法。
- 演示 BFS、DFS、最小生成树和最短路径算法。
- 设计和测试随机图或特定图结构。
- 数据可视化和算法过程验证。
