# 排序模式介绍

## 1. 概述
排序模式是 SortWidget 中用于展示和执行各种排序算法的功能模块。  
用户可以通过图形化界面选择排序算法、调整排序顺序、控制动画速度，并可对数据进行插入、删除、更新或查找操作。

主要功能：
- 可视化排序过程
- 支持多种排序算法（冒泡、插入、选择、快速、归并、堆排序）
- 支持升序/降序切换
- 自动运行和手动步进操作
- 数据节点增删改查

---

## 2. 排序方法

| 算法 | 功能说明 |
|------|----------|
| Bubble Sort | 冒泡排序，逐步交换相邻元素 |
| Insertion Sort | 插入排序，将元素插入到已排序部分 |
| Selection Sort | 选择排序，每次选择最小（或最大）元素 |
| Quick Sort | 快速排序，分治策略，效率较高 |
| Merge Sort | 归并排序，将数组拆分后再合并 |
| Heap Sort | 堆排序，利用二叉堆进行排序 |

---

## 3. 使用方法

### 3.1 选择排序算法
通过下拉框选择排序算法：
1. 确保上一次排序已完成
2. 点击下拉框选择所需排序方法
3. 系统会自动初始化数据和排序状态

### 3.2 数据操作
- **插入节点**：在指定位置插入整数值
- **删除节点**：删除当前高亮的节点
- **更新节点**：修改当前高亮节点的值
- **查找节点**：输入数值，系统将高亮第一个匹配的节点

### 3.3 排序控制
- **手动步进**：点击 `Next` 按钮执行下一步操作，`Last` 回退上一步
- **自动播放**：点击 `Auto` 按钮自动播放排序动画，可通过滑动条调整速度
- **排序方向**：通过 `setAscending` 切换升序或降序

### 3.4 数据管理
- **随机生成数据**：快速生成随机整数序列
- **保存数据**：将当前数据序列保存到文件
- **加载数据**：从文件加载数据序列

## 4. 使用示例

### 4.1 快速排序示例
1. 点击下拉框选择 `Quick Sort`
2. 点击 `Auto` 自动播放排序
3. 可通过滑动条调整动画速度

### 4.2 数据节点操作示例
- Insert: 在位置 2 插入值 10
- Remove: 删除高亮节点
- Update: 将高亮节点值修改为 7
- Find: 查找值为 5 的节点并高亮

### 4.3 手动控制示例
- 点击 `Next` 查看下一步交换或更新操作
- 点击 `Last` 回退到上一步
- 自动播放时可随时点击 `Auto` 停止

---

## 5. 注意事项
1. 在选择排序算法前，请确保上一次排序已完成
2. 数据操作的索引必须有效，否则系统会提示错误
3. 自动播放时可以调整速度，但不能同时执行手动步进
4. 堆排序和归并排序会以特殊图形方式显示（堆结构或分区条）
